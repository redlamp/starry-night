# Streets of Rage — Sega Genesis Intro Screen (1991)

![[streets-of-rage-intro.webp]]

**Date adopted as direct reference:** 2026-05-21

The tile-able skyline backdrop from the original *Streets of Rage* (Bare Knuckle in Japan), Sega Genesis, 1991. Designed to scroll horizontally during the intro. Established a visual template that anchors this project.

## Palette (sampled by eye)

| Element | Approx | Notes |
|---|---|---|
| Sky | `#0a1838` deep navy | Not pure black — slight blue holds the world together |
| Mid-band tall tower body | `#3a7da0` / `#4a9bb8` teal-cyan | These towers *glow*; not just lit windows, the whole building face emits |
| Foreground silhouette | `#0a0a14` near-black | Pure silhouette, only sparse warm windows |
| Distant back row | `#10151f` darker navy | Reads as receding mass, almost no windows |
| Warm window | `#f0c060` / `#e8a040` yellow-amber | The window color. No cool fluorescents in this image |
| Bright warm | `#f8d070` | Brightest pixels — streetlights and a few standout windows |
| Streetlight glow | `#ffd060` | Yellow round dots at base, evenly spaced |
| Road dashes | `#c84020` orange-red | Horizontal lines at bottom strip — traffic and lane markings |

## Composition rules

- **Three depth layers**, not five: foreground silhouettes (close, dark) → mid teal towers (the heroes) → distant flat back row
- **Strong horizontal banding**: sky / mid tower band / dark foreground / road strip
- **Symmetric tile-ready**: the image loops as a scrolling backdrop, so silhouettes repeat
- **Density gradient**: window count rises sharply on mid towers, sparse on foreground, almost none on back row
- **Bottom 20-25% is the street layer**: streetlights + road dashes, not buildings

## What we take

- Sky moves from `#000` → deep navy `#0a1838`
- Building palette splits by row, not by archetype only:
  - Front row: near-black silhouette, sparse warm windows
  - Mid rows: teal-cyan facade *with body glow* (slight emissive), dense yellow windows
  - Back row: dark navy, almost no windows
- Window palette becomes warm-only (yellow + amber + dim amber + TV-blue flicker). The "office cool" cyan I had earlier doesn't match this source — drop it for now or push to backlog as a future style variant.
- Backlog confirmed: streetlights at ground line, road dash layer
