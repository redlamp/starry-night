Long grilling session (`/grill-with-docs`, ultracode) on city-planning realism — the road/highway system and building placement read wrong from above. No code changed; this was a design + docs day.

## Diagnosis (from reading the generators + two agent workflows)

- The radial **starburst** the user saw is `arterials.ts:80-91`: 6–8 arterial spokes from the most-central district to the map edge, in **100% of seeds, every topology**. Not the 10% ring-radial highway case. Violates the streets-first note's own arterial spec.
- Only **Stage 1** of [[decision-streets-first-city-generation]] shipped, and it diverged: districts are Voronoi cells with independent rotated grids, roads bolted on top, buildings deleted where they collide. **Streets-last, not streets-first.** Blocks never bounded by roads; buildings 15–30 m off the street.
- The team had earlier de-weighted ring *highways* (0.20→0.10) believing "too many rings" — fixed the wrong layer; the starburst is the arterials.

## Decided → [[decision-grid-first-city-generation]]

Grid-first rework. Highlights:
- Coherent **grid substrate**; districts are **zones over the grid**, differing by size/density/zone/height, mostly not orientation.
- **Patchwork** via a **center-anchored orientation field** (θ₀ locked at center, drift outward = tunable knob) sampled per district → neighbours mostly similar.
- **Continuous seam streets**: Voronoi cell edges become real streets each grid T's into — chosen candidate **B** (realism) but with workflow candidate **A**'s seam guarantee, so it can't cross-hatch.
- **Lines-first invariant**: roads are continuous lines; intersections/blocks are emergent overlaps; no isolated intersection placed.
- **Arterials = heavy grid lines** (every Nth) + promoted seam runs. Starburst deleted.
- **Legacy diagonals** (0–3) for diagonal interest; **highways** laid last.
- **Voronoi metric** L2 → **Chebyshev L∞** in the core frame (rectilinear, map-like seams).
- **Zone field** anchored to road hierarchy (commercial on arterials/center, industrial on highways/edges, residential interiors).
- **Zone-driven hybrid fill** (perimeter wall + towers / lot rows / big parcels) at a small **sidewalk setback**.
- **Wedge/flatiron** = 4-sided trapezoidal box, **downtown only**. No-new-mesh rule relaxed to "a few shared instanced shapes."
