### What got locked

- **Goal statement** — "different seeds = categorically different cities," readable from slow-orbit camera distance
- **Generative inversion** — streets-first hybrid (Path C, B-weighted): highway + arterial network laid down first, district shells derived from network closure, buildings emerge from network geometry
- **Glossary** — created [[CONTEXT.md]] (project-root glossary file). Retired *cluster* + *featured cluster* in favour of:
  - **District** (the bigger perceptual area) with 6 characters: Downtown high-rise / Subcentre high-rise / Heritage / Residential / Industrial / Mixed-use
  - **Zone** (polygon-level use class): Residential / Commercial / Industrial / Mixed-use
- **Network**: Highway (1–2 per seed) + Arterial (3–6 per seed) + implicit Local street boundaries
- **Topology library**: Crossroads (0.35) / Bypass (0.25) / Ring (0.20) / Ring + radial (0.20)
- **Silhouette templates** for high-rise districts: Tabletop (0.45) / Wedding cake (0.30) / Twin-peak (0.15) / Landmark (0.10)
- **Height-cap matrix** — `zone × district-character → max height m`; silhouette template applies as height-multiplier field on top
- **Block + lot + setback rules** with per-district regularity slider (Downtown 0.10 strict → Heritage 0.85 organic)
- **Streetlight planning** — modern-LED defaults, per-tier (highway 4000K uniform; arterial 4000K uniform; local per-zone), 4% variant-bulb-chance, failure flicker reuses window TV-flicker shader path
- **District naming** — cardinal + character display name + stable ID
- **Industrial / high-rise interface** — lean Mixed-use for Stage 1, `transitional-bias` flag in Stage 2
- **District-merging-across-highways** — strict in Stage 1, Stage 2 cheap name-collapse pass
- **Heritage adjacency buffer** replaces literal view-corridor rules (no fixed viewpoint exists for slow-orbit camera)
- **Podium-and-tower** archetype + per-seed *Pacific Rim flag* — Stage 2/3 addition
