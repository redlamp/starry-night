## Play/pause wiring + per-model transport defaults (this is the load-bearing bit)
- The Orbit header play button / Space / controls-guide all write one `orbitPaused` flag.
  Drift + Turntable were keeping **local** `paused` refs, so the button never reached
  them. Both now read the shared flag → **one transport control governs Map, Drift,
  Turntable**.
- Per-model default starting state via `CameraModelMeta.startsPaused`, applied on every
  *user* switch (selector + `t`/`f` hotkeys), **not** in the generic setter (so hydration
  still restores saved poses exactly). **Map** starts paused (curated still pose);
  **Drift / Turntable** auto-play. Initial `orbitPaused` flipped `true → false` (default
  model is now Drift). Detail in [[decision-camera-model-registry]].
