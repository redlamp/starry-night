## Day 2 — M1 ticket 2-3 + tooling + districts

Carried over from [[2026-05-21]] in one continuous dev session.

### What landed

#### M1 ticket 1 — scaffold (committed yesterday)

- Next.js 16 + Bun + R3F v9 + drei v10 + Three.js r184
- Tailwind v4, ESLint flat config, Prettier
- Initial scene: black bg, drei Stars, single placeholder building

#### M1 ticket 2 — seeded multi-building city

- `lib/seed/cityGen.ts` — first pass: 3 rows of buildings, then expanded to grid, then rebuilt into district-based layout
- 7 archetypes — `low-rise`, `warehouse`, `mid-rise`, `residential-tower`, `narrow-tower`, `office-block`, `spire`
- Per-archetype dimension ranges + per-building jitter (W ±20 %, D ±15 %, H ±25 %, 5 % outlier)
- Downtown elliptical bias zone — height boost up to 1.7×, smooth radial falloff

#### M1 ticket 3 — shader-painted windows

- `lib/shaders/window.ts` — fragment shader reads per-window state from a `DataTexture`, paints window grid on building side facades
- `lib/seed/lightingGen.ts` — window state generator with kelvin-colour palette (1800–6500 K)
- `lib/color/kelvin.ts` — Tanner Helland kelvin→RGB approximation
- Window cell-fill `0.3 × 0.5` (taller than wide, like apartment windows)
- Layer-aware: front near-black silhouette, mid teal body-glow, back deep navy

#### District-based city layout

- 8 districts: `downtown`, `midtown`, `oldtown`, `residential-west/east`, `commercial-bridge`, `industrial-south`, `harbor-east`
- Each district has world position, rotation, block sizes, street widths, character bias, block jitter
- Buildings generated in district's local rotated frame, transformed to world
- Each building gets `rotationY` field so it faces its stripe — camera-orbit ready
- Downtown ellipse boost still bleeds across district seams

#### UI tooling

- `components/ui/CameraPanel.tsx` — editable position / lookAt / rotation / fov, mode toggle (still ↔ fly), live readout, perf section, seed input + reroll
- `components/scene/CameraControls.tsx` — handles still-mode camera intent + fly mode (PointerLockControls + custom WASD/Space/C/Q/E)
- Tween between presets with easeInOutCubic, 900 ms — `Default` and `Top-down` presets
- `components/scene/PerfMonitor.tsx` — fps, triangles, draw calls, geometries, textures, updated 2 Hz

#### Atmosphere

- Linear fog `220..1100 m` matched to sky colour
- `components/scene/Streetlights.tsx` — additive Points cloud, custom shader for soft disc + size attenuation, deterministic placement at block corners per-district
- `components/scene/Moon.tsx` — disc at `(-180, 320, -500)`, radius 22 m
- `components/scene/Ground.tsx` — 4000 × 4000 m plane, `#04060c`
